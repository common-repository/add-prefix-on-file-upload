Add a custom prefix after uploading a file

== Description ==

This plugin allows you to automatically add a custom prefix after uploading a file.


### ✅ **Custom prefix** ✅

The "Add Prefix on File Upload" plugin is a useful tool that allows you to effortlessly add a custom prefix to any file you upload. This feature can be extremely useful for organizing your files, as well as providing additional context to the contents of your uploads. With just a few clicks, you can customize your file names to better reflect their purpose and contents. 

Whether you're a blogger, website owner, or content creator, this plugin is an essential tool for streamlining your file upload process and ensuring that your files are always organized and easily accessible.


**Features:**

* Automatically adds a custom prefix after uploading a file
* Easy to use and customize
* GPL licensed
* 100% Free
* No ads



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/add-prefix-file-upload` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings > Add Prefix on File Upload screen to configure the plugin


== Frequently Asked Questions ==

= How do I change the prefix for uploaded files? =

To change the prefix, go to the Settings > Add Prefix on File Upload screen and change the "New File Name Prefix" field.
